## ゲーム概要

起きている猫をクリック/タップして、**全員を寝かせたらクリア**。  
ただし、寝た猫に触れると別の猫が起きてしまうことも。手順を考えて解こう！

---

## 操作方法

### マウス / タッチ
- **左クリック / タップ**：猫を切り替え（起きる⇔寝る）
- **右上の ↺**：リトライ

### キーボード（PC）
- **R**：リトライ
- **Esc**：メニュー
